# Fake News Generator (Python)

## About the Project
This project is a simple Python script that generates fictional and clearly fake news headlines for educational and learning purposes. It demonstrates the use of lists, random selection, and basic text generation in Python.

## Features
- Generates random fake news headlines
- Beginner-friendly code structure
- Uses Python's built-in random module
- Easy to customize and expand

## How It Works
The program randomly selects:
- A subject
- An action
- An object
- An additional detail

These elements are combined to form a complete fictional news headline.

## Technologies Used
- Python 3
